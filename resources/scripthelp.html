Script help

Welcome to the computer scripter. This is how you will make each network node do something. Note that each IP should be a string in the format of "x.x.x.x", each port should be an integer, and that each bit of packet data should be a string.

There are many classes provided to you:

Utility

This class is a pre-instantiated global object that provides basic utility functions.

  • getIP() Returns the current computer's IP
  • getBroadcastAddr() Returns the netwrok's broadcast address
  • pingIP(ip) Returns true/false whether ip exists.

UDP

This class is a pre-instantiated global object that allows the sending and receiving of UDP packets.

  • send(ip, port, data) Sends data to ip:port.
  • onReceive(func(ip, port, data)) Sets func to be called when the computer receives a UDP packet. func should have three parameters, ip, port, and data.

TCP

This class is a pre-instantiated global object that allows the establishment of TCP clients and servers.

  • createServer(port) Creates a TCP server at port and returns the TCPConnection object that represents the server.
  • connectToServer(ip, port) Connects to a TCP server at ip:port and returns the TCPConnection object that represents the client.
  • getServerAtPort(port) If you created a server at port, return it. Else return null.
  • getClientAtIPAndPort(ip, port) If you connected to a server at ip:port, return the client object. Else return null.

TCPConnection

This is the class that is returned by TCP methods such as createServer() and connectToServer(). It can represent either a client or a server object.

  • isServer() Returns true if server, false if client.
  • onConnection(func(remote)) Sets the callback function, func, for whenever a new TCP connection is established.
  • Server getClients() Returns a list of connected clients.

Sample programs

Print IP

print(Utility.getIP())

Broadcast UDP packet to network

var port = 300; //can be any integer UDP.send(Utility.getBroadcastAddr(), port, "Hello!");
